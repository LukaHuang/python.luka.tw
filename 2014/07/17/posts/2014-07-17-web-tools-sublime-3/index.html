Sublime 3 套件整理

學習網頁一直以來有個困擾，打括號真的是慢到不行，原來sublime這個文字編輯器有許多方便的套件可以用。用簡單的指令就可以架構出一個網站。

  2. ConvertToUTF8 ： 解決 Sublime Text 3 預設不支援 Big5 與其他 CJK 編碼的問題
